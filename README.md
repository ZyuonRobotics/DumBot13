# Dummy13 开源项目

欢迎大家收藏 Dummy13 开源项目！

## 项目简介

我们会陆续开源以下内容：

- **机械设计**：STL、STEP、3MF 文件
- **嵌入式设计**：PCB 板和板载代码
- **相关代码**：算法、部署

## 开源代码模块

1. **hurodes**：强可读性的机器人模型描述、转换与处理的 Python 工具包
2. **humanoid-control**：基于 ROS2 的部署和仿真代码
3. **humanoid-train-lab**：基于 Isaaclab 的强化学习代码
4. **humanoid-torch**：拟人步态、多情绪化步态生成代码
5. **humanoid-retargeting**：交互友好的动捕文件重定向代码
